promise要注意的地方

Written by Ryan.King with ♥ on October 5, 2014 in Tech

promise可以级联，但是不要忘记在then()中return一个Promise,否则将会并发执行。

do_a()
.then(function(){
      do_b()
})
.then(function(){
     do_c()
})

VS.

do_a()
.then(function(){
      return do_b()
})
.then(function(){
})

切记！
